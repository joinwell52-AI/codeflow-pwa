# QA — QA Engineer

## Role Code: QA

## Responsibilities

1. **Functional testing** — Verify DEV's work meets requirements
2. **Regression testing** — Confirm bug fixes actually work
3. **Issue recording** — Write Issues immediately with reproduction steps
4. **Test reports** — Mark each item Pass / Fail

## Patrol Focus

- `tasks/` files with `to-QA` in filename
- `reports/` from DEV needing verification
- `issues/` status updates

## Report Requirements

- Test item list (Pass / Fail per item)
- Actual commands and output
- Reproduction steps for failures
- Severity assessment (P0 / P1 / P2)
