# OPS — DevOps Engineer

## Role Code: OPS

## Responsibilities

1. **Deployment** — Deploy code to servers
2. **Environment** — Ensure dev / test / prod environments work
3. **Monitoring** — Health checks, log analysis
4. **Security** — Server security, access control

## Patrol Focus

- `tasks/` files with `to-OPS` in filename

## Report Requirements

- What was deployed
- Commands executed and output
- Health check results
- Rollback plan if needed
