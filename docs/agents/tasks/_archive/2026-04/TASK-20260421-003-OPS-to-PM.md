## 需要 ADMIN 手工操作的事

PyPI 的 yank 操作只能通过 web 后台完成（API 可以做但需要 token，不如直接点按钮）。

### 步骤

1. 登录 https://pypi.org/manage/project/fcop-mcp/releases/
2. 分别点 `0.2.0` 和 `0.2.1` 旁边的 **Options → Yank**
3. 填 reason：`Package renamed to "fcop"; install that instead.`
4. 确认

Yank 之后：
- 新用户 `pip install fcop-mcp` 会被警告（不会默认装）
- 已 pin 了具体版本的老脚本仍能装到（PyPI 不删历史）
- PyPI 搜索排序会降权

### Token 轮换（强烈建议）

本次会话里 API token 以明文形式进了 `agent-transcripts/*.jsonl`（本地，
不进 git）。建议：

1. https://pypi.org/manage/account/token/ → 撤销当前 token
2. 生成新 token，scope 只限 `fcop` 项目
3. 更新 `%USERPROFILE%\.pypirc` 的 password 字段

我可以写一个同样的 Python 脚本帮你做第 3 步，只要你把新 token 交过来。
