## 遗留风险 / B 计划

- **PyPI 名字被抢注**：上传瞬间 `codeflow-mcp` 若已被他人占，`twine upload`
  会 400。B 计划包名候选：`codeflow-fcop`、`bridgeflow-mcp`、`fcop-mcp`。
  改名代价：只改 `pyproject.toml` 的 `name` 字段 + `[project.scripts]` 那行的
  左值，一次 `py -3.10 -m build` 即可重出包。**Python 包名（`codeflow_mcp`）
  保持不变**，用户侧零感知。
- **`fastmcp` 启动时的双重注册 WARNING**（`Component already exists:
  resource:codeflow://status`）：`__init__.py` 做了 `from .server import main`，
  命令行入口再 `import codeflow_mcp.server` 时会走第二次装饰器；装饰器在
  FastMCP 里是幂等的（只打 WARNING，不重复注册），但观感略吵。**不阻塞发布**，
  后续可把 `__init__.py` 的 re-export 换成 lazy `__getattr__`。
- **PyPI metadata 里的 `Homepage` URL** 当前指向 `https://github.com/joinwell52/bridgeflow`
  （这是猜的——ADMIN 的 PyPI 用户名是 `joinwell52`，但 GitHub 侧实际仓库地址
  没核对过）。如果 GitHub 没这个仓库，PyPI 项目页链接会 404。**OPS 上传前请
  ADMIN 确认 URL**，不对就改 `pyproject.toml` 再重 build。
