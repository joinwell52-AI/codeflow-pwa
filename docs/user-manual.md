# BridgeFlow 用户操作手册

**版本：** v0.2.1 ｜ **更新日期：** 2026-04-02

---

## 第一次使用（全程约 5 分钟）

### 前提：安装 Python

BridgeFlow 依赖 Python 3.10+。如果你的电脑已有 Python，跳过此步。

**Windows 安装 Python：**

1. 打开 https://www.python.org/downloads/
2. 下载最新版本（3.10 或以上）
3. 运行安装程序，**务必勾选 "Add Python to PATH"**
4. 安装完成

> 不确定是否已安装？打开命令提示符（Win+R 输入 cmd），运行 `python --version`，显示版本号说明已安装。

---

### 前提：在 Cursor 中建立 4 个 Agent

BridgeFlow 需要 Cursor 中存在以下 4 个 Agent，**名称必须完全一致**：

| 编号 | Agent 名称 | 角色职责 |
|---|---|---|
| 1 | `1-PM` | 项目经理，接收任务、拆解调度 |
| 2 | `2-DEV` | 开发工程师，负责编码实现 |
| 3 | `3-QA` | 测试工程师，负责验收回归 |
| 4 | `4-OPS` | 运维工程师，负责部署发布 |

**操作步骤：**

1. 打开 Cursor
2. 点击 Chat 面板右上角 **「+」** 新建 Agent
3. 点击 Agent 名称，将其重命名为 `1-PM`
4. 重复上述步骤，依次创建 `2-DEV`、`3-QA`、`4-OPS`

> **注意：** 名称中的数字编号和大小写必须完全一致，BridgeFlow 启动时会自动检测，
> 任何一个缺失都会提示并退出。

---

### 第一步：下载一键启动脚本

从以下地址下载 `bfstart.bat`（Windows）：

```
https://github.com/joinwell52-ai/BridgeFlow/raw/main/scripts/bfstart.bat
```

**把这个文件放在你想要存放任务文件的项目目录里**，例如 `C:\my-ai-team\` 或 `D:\my-ai-team\`，任意盘符均可。

---

### 第二步：双击运行

双击 `bfstart.bat`，脚本会自动完成：

```
✓ 检查 Python 版本
✓ 安装 BridgeFlow（首次约 10-30 秒）
✓ 自动初始化配置（生成唯一房间号）
✓ 启动 BridgeFlow
✓ 自动打开浏览器仪表盘
```

运行完成后，浏览器会自动打开 `http://localhost:18765`：

```
┌─────────────────────────────────────┐
│  BridgeFlow 仪表盘                  │
│                                     │
│  环境：✓ Python  ✓ Cursor           │
│  中继：● 已连接                     │
│  房间：bf-my-pc-a3f2c8d1           │
│                                     │
│       ┌──────────────┐              │
│       │  [  QR码  ]  │  ← 手机扫这里
│       └──────────────┘              │
└─────────────────────────────────────┘
```

---

### 第三步：手机扫码绑定

1. 用手机浏览器打开 PWA：

   ```
   https://joinwell52-ai.github.io/bridgeflow-pwa/
   ```

2. 建议添加到主屏幕（像 App 一样使用）：
   - iOS Safari：点底部分享按钮 → "添加到主屏幕"
   - Android Chrome：右上角菜单 → "添加到主屏幕"

3. 打开 PWA，点顶部右角 **「我的」**

4. 点 **「📷 扫码绑定 PC」**，对准仪表盘上的二维码

5. 扫描成功后，手机顶部连接灯变绿 ✅

**绑定完成！** 手机和 PC 现在互相认识了，房间号也自动同步。

此时你的目录里会多出一个文件：

## 日常使用（每次启动只需 1 步）

**双击 `bfstart.bat`** — 结束。

脚本会自动：
- 检查 Python 和 Cursor 环境
- 检查并升级到最新版本
- 启动 BridgeFlow
- 打开浏览器仪表盘
- 连接中继，等待手机指令

---

## 发送任务

手机 PWA 底部发送区：

```
1. 选择接收角色：PM / DEV / OPS / QA
2. 输入任务内容
3. 选择优先级：P0（紧急）/ P1（高）/ P2（中）/ P3（低）
4. 点「发 送」
```

任务发出后，PC 端的 AI 角色会收到任务并开始处理。

---

## 查看回复

- 在 PWA 主页任务列表，点任意任务卡片展开，可以看到完整对话记录
- 顶部"今日回复"数字有变化说明有新回复
- 手机会收到 AI 的回复摘要推送

## 遇到问题

### 仪表盘连接灯是红色

PC 端无法连接中继服务器。检查：
1. 网络是否正常
2. 防火墙是否拦截了 `wss://ai.chedian.cc`
3. 关闭后重新双击 `bfstart.bat`

### 手机扫码没反应

1. 确认已授予浏览器摄像头权限
2. 确认手机 PWA 顶部连接灯是绿色（先连上中继再扫码）
3. 刷新仪表盘页面，重新扫码

### PWA 页面没更新

清除手机浏览器缓存：
- Safari：设置 → Safari → 清除历史记录与网站数据
- Chrome：设置 → 隐私设置 → 清除浏览数据

### 发现有新版本

重新双击 `bfstart.bat` 时，如果检测到新版本，会询问：

```
  🚀 发现新版本 v0.2.1（当前 v0.2.0）
  是否立即升级并重启？[y/N]
```

输入 `y` 回车，自动升级并重启，无需其他操作。

## 附录

### 我的房间号在哪里？

双击 `bfstart.bat` 启动时，命令行窗口会显示：

```
  房间号    : bf-my-pc-a3f2c8d1
```

浏览器仪表盘（`http://localhost:18765`）也会显示。

手机端"我的"页面也能看到当前房间号。

### 多台 PC 怎么用？

每台 PC 上放一份 `bfstart.bat`，双击运行，各自自动生成独立配置和唯一房间号。手机分别扫码绑定各台 PC 即可。

### 任务文件存在哪里？

```
你的项目目录/
  docs/agents/tasks/      ← 任务文件（TASK-*.md）
  docs/agents/reports/    ← 回执文件
```

每条任务都以标准 Markdown 文件保存，可以用任意编辑器打开查看。
