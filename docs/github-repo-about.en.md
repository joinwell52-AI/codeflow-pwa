# GitHub Repository Page (About) Setup

Applies to **`joinwell52-AI/codeflow-pwa`**. The web UI cannot be written automatically by scripts in this repo; please save the settings once on GitHub manually.

## Where to Go

1. Open: <https://github.com/joinwell52-AI/codeflow-pwa>
2. In the **About** section on the right → click **⚙ (Edit repository details)**  
   Or: **Settings** → **General** → **Repository name / Description** at the top.

## Suggested Values

### Description (one line)

**Bilingual (recommended, better for search):**

```text
CodeFlow / 码流 — Multi-agent collaboration via file-driven TASK protocol; mobile PWA + PC desktop.
```

**Chinese only:**

```text
码流（CodeFlow）— 文件驱动多角色协作；手机 PWA + PC 桌面端，TASK 文件名即协议。
```

### Website

```text
https://joinwell52-ai.github.io/codeflow-pwa/
```

### Topics

In the **Topics** field of the About editor, add tags one by one (press Enter to confirm). Suggested:

`codeflow` `pwa` `cursor` `mcp` `websocket` `multi-agent` `collaboration` `task-protocol` `agent-bridge`

(You may trim the list as needed; not all are required.)

## Relation to the Old Name `bridgeflow-pwa`

The repository was renamed from **`bridgeflow-pwa`** to **`codeflow-pwa`**. If the Description still shows the old name, overwrite and save using **Description** above; it does **not** update automatically from the README.
