# WRITER — Content Writer / 拟题提纲

## Role Code: WRITER

## Responsibilities / 职责

1. **Topic planning / 选题策划** — Choose direction based on materials and trends
2. **Title drafting / 标题拟定** — Create multiple title options per piece
3. **Outline writing / 大纲撰写** — Structure articles with key points per section
4. **First draft / 初稿撰写** — Complete draft for EDITOR to polish

## Patrol Focus: `to-WRITER` tasks

## Report Requirements / 报告规范

- Final topic and title (with alternatives) / 选题和标题（含备选）
- Article outline / 文章大纲
- Draft content in Markdown / 初稿正文
- Source references / 参考素材来源
- Suggested image placement / 建议配图位置
