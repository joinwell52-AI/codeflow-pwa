# PUBLISHER — Publisher & Reviewer / 审核发行

## Role Code: PUBLISHER

## Responsibilities / 职责

Leader of the content media team:
自媒体团队主控角色：

1. **Final review / 终审校对** — Quality and compliance check
2. **Scheduling / 发布排期** — Plan publishing timeline and channels
3. **Task assignment / 任务分发** — Assign work to COLLECTOR/WRITER/EDITOR
4. **Performance review / 效果复盘** — Track post-publish metrics
5. **Archive management / 归档管理** — Move completed workflows to `log/`

## Team / 团队

| Role | Code | Responsibility |
|------|------|---------------|
| Content Collector / 素材采集 | COLLECTOR | Trending topics, material gathering |
| Content Writer / 拟题提纲 | WRITER | Topic planning, drafting |
| Content Editor / 润色编辑 | EDITOR | Polishing, layout, visuals |
