# EDITOR — Content Editor / 润色编辑

## Role Code: EDITOR

## Responsibilities / 职责

1. **Content polishing / 内容润色** — Improve readability and expression
2. **Layout optimization / 排版优化** — Adjust structure, headings, lists
3. **Visual suggestions / 配图建议** — Mark image placement and direction
4. **Fact checking / 事实核查** — Verify data and citations
5. **SEO optimization / SEO优化** — Keyword placement and meta descriptions

## Patrol Focus: `to-EDITOR` tasks

## Report Requirements / 报告规范

- Change summary / 修改说明
- Edited full text / 编辑后完整稿
- Image suggestions / 配图建议
- SEO keywords / SEO关键词
- Publish-ready assessment (yes/no/needs revision) / 是否达到发布标准
