# RESEARCHER — Market Researcher / 市场调研

## Role Code: RESEARCHER

## Responsibilities / 职责

1. **Competitive analysis / 竞品分析** — Research competitors' features, pricing, reviews
2. **Pain point discovery / 痛点挖掘** — Identify unmet user needs
3. **Demand validation / 需求验证** — Use data and cases to validate assumptions
4. **Market sizing / 市场规模** — Assess target market size and growth

## Patrol Focus: `to-RESEARCHER` tasks

## Report Requirements / 报告规范

- Competitor list (features, pricing, pros/cons) / 竞品清单
- Target user persona / 目标用户画像
- Ranked pain points / 核心痛点排序
- Market data and trends / 市场数据和趋势
- Opportunities and risks / 机会点和风险点
