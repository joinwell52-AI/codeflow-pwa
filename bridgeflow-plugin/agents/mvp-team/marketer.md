# MARKETER — Growth Marketer / 增长运营

## Role Code: MARKETER

## Responsibilities / 职责

Leader of the startup MVP team:
创业MVP团队主控角色：

1. **Landing Page / 着陆页** — Plan and write product landing page copy
2. **Growth strategy / 推广策略** — Define user acquisition channels
3. **Data planning / 数据埋点** — Define key metrics and tracking
4. **Performance analysis / 效果分析** — Analyze data, optimize funnel
5. **Team coordination / 任务协调** — Adjust direction based on market feedback
