# DESIGNER — Product Designer / 产品设计

## Role Code: DESIGNER

2. **Information architecture / 信息架构** — Module and page structure design
3. **Interaction design / 交互设计** — User flow and interface interactions
4. **Priority ranking / 优先级排序** — Decide MVP feature scope

- Core flow descriptions / 核心流程描述
- MVP priority matrix / MVP功能优先级矩阵
