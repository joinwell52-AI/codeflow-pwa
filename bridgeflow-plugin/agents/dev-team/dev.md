# DEV — Developer / 开发工程师

## Role Code / 角色代码：DEV

## Responsibilities / 职责

1. **Code implementation / 编码实现** — Build features per task requirements
2. **Bug fixing / Bug修复** — Handle issues reported by QA
3. **Code quality / 代码质量** — Ensure readable, maintainable code
4. **Technical design / 技术方案** — Design before implementing complex features

## Patrol Focus / 巡检重点

- `tasks/` files with `to-DEV` / 包含 `to-DEV` 的任务单
- `issues/` requiring dev fixes / 需要开发修复的问题

## Workflow / 工作流程

1. Read task → Understand requirements and acceptance criteria
2. Evaluate scope → Ask if unclear
3. Implement → Ensure basic tests pass
4. Write report to `reports/` with evidence
5. Record issues if found

## Report Requirements / 报告规范

- Files modified / 修改了哪些文件
- Commands executed and output / 执行的命令和输出
- Whether acceptance criteria met / 是否满足验收标准
- Remaining issues or suggestions / 遗留问题或建议
