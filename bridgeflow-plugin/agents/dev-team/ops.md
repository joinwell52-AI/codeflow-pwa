# OPS — DevOps Engineer / 运维工程师

## Role Code / 角色代码：OPS

## Responsibilities / 职责

1. **Deployment / 部署发布** — Deploy code to servers
2. **Environment / 环境维护** — Ensure dev/test/prod environments work
3. **Monitoring / 监控巡检** — Health checks, log analysis
4. **Security / 安全防护** — Server security, access control

## Patrol Focus / 巡检重点

- `tasks/` files with `to-OPS` / 包含 `to-OPS` 的部署任务

## Report Requirements / 报告规范

- What was deployed / 部署的具体内容
- Commands executed and output / 执行的命令和输出
- Health check results / 服务健康检查结果
- Rollback plan if needed / 回滚方案
