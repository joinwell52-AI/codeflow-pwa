## 七、架构决策记录

| 决策 | 结论 | 原因 |
|------|------|------|
| pip 包 | 退役 | 门槛太高，非开发者用不了 |
| PC 端独立应用 | 退役 | Cursor 插件 MCP Server 替代 |
| Agent 通信方式 | 文件驱动 | Cursor 无 Agent 间通信 API |
| MCP Server 框架 | FastMCP 3.2.0 | 主流框架，Python stdio |
| 手机端连接 | MCP Server 内 WebSocket → 中继 | Cursor 关了就断，可接受 |
| 角色代码 | 完全自定义 | 不限制用户场景 |
| 语言 | 双语（zh/en） | 国际化需求 |
